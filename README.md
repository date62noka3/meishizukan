# 名刺図鑑

名刺図鑑は名刺を管理するアプリであり、  
社会人が人脈を有効利用できるよう支援するアプリです。  
名刺や名刺入れにまつわる様々な問題を解決します。

# DEMO(プロモーション)
動画を是非ご視聴ください。  
https://youtu.be/Wfu3Pb-ho6Q
 
# Description
新社会人の不安を払しょくすることもできると考えています。  
実際、私もあと1年したら新社会人ですが、目上の方の名前や顔を正確に覚えられるか不安です。  
ちゃんと覚えていると、好感を持ってもらえると思いますし、信頼関係を築けると思います。  
でも現実、すぐには覚えられない。だからアプリに頼っていいんです。  
名刺図鑑があれば、事前に素早くスマホで確認することができます。  

【使用したライブラリ】  
Coroutine, Admob, Junit
